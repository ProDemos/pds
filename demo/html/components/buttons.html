Buttons

Button

Text button, large (=default)

Large is the default style of a button if no other properties are given

{{ ui.component('buttons/text', {
    "text": "Button",
    "url": "#",
    "theme": "corporate",
    "level": "primary",
    "size": "large"
}) }}
<a
    class="c-btn  c-btn--primary c-btn--corporate c-btn--large  "
    data-component="button" href="#">
        
    <span class="c-button__icon-container__label">Button</span>
</a>
