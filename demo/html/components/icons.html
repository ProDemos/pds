Icons

List of all kind of icons used throughout the project. Every icons starts with a class (pds-c-icon) and a modifier to specify which icon. Optional additional modifiers are colors, 'colored' and 'accented' to use the current themes color and accent, and 'large'.


Elements and modifiers

  • .pds-c-icon [.{name}] [.large] [.colored] [.accented] [.{color}]
