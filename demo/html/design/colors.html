Colors

The design system has a set of named colors, that you can access through css modifiers (pds-m-foo.blue) or using sass (@pds-color(blue)).


These colors are grouped in themes that you can access through css themes (pds-t-corporate) or using sass (@pds-theme-color(variant01,stroke-accent)).
