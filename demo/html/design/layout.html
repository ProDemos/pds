Layout

Spacings

The design system has a set of numbered spacings that you can access in sass (nth($pds-spacings, 4)).


Breakpoints

The design system has a set of named breakpoints that you can access in sass. The names are
xs, sm, md, lg, xl It contains some helpers mixins:

@include media-until(xs) { h1 { color:red; } } @include media-between(xs,md) { h1 { color:green; } } @include media-from(md) { h1 { color:yellow; } }
